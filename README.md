# To-Do List App

A simple to-do list application built with React and Redux for managing tasks. The app allows users to add, edit, delete, and mark tasks as completed.

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Redux state management

## Technologies

- React.js
- Redux
